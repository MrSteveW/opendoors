# Opendoors v4 booking app

## Features

- [x] Next.JS with PostgreSQL database
- [x] Calendar by FullCalendar
- [x] Authentication and Authorization by Clerk

## Additional libraries used

- FullCalendar
- pg database interface
- Zustand state management
- Clerk
- Lucide icons
- Motion
